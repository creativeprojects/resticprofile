Pre-built binaries for resticprofile are available for various operating system and architectures.
You can download the latest versions from the [releases page](https://github.com/creativeprojects/resticprofile/releases).

## Pre-built binaries available:
- windows
    - 386
    - amd64
    - arm64
- linux
    - 386
    - amd64
    - arm64
    - armv6
    - armv7
    - mips64 hardfloat
    - mips64le hardfloat
    - mips hardfloat
    - mipsle hardfloat
    - ppc64
    - ppc64le
    - riscv64
    - s390x
- darwin
    - amd64
    - arm64
- freebsd
    - 386
    - amd64
    - armv6
    - armv7
- openbsd
    - 386
    - amd64
- netbsd/amd64
- solaris/amd64
